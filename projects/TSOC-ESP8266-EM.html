thingSoC : ESP8266 Embedded Module Reference Design

The TSOC-ESP8266-EM board is a complete Wi-Fi Device and Access point, supported by the Arduino IDE. It places the popular ESP8266 Wi-FI System-on-Chip Device in a thingSoC Embedded Module Socket, and enables the ESP8266 to be used with any other thingSoC reference design, allowing it to be paired with a number of different host processors and peripherals.

TSOC-ESP8266-EM Features:

  • ESP8266 WiFi System-on-Chip
  • Program using the Arduino IDE
  • Program using Node.js
  • Program using eLua
  • Flash programming using RTS/DTR, compatible with nodeMCU
  • Inter-Integrated Circuit(I2C) Bus supported
  • Serial-Parallel Interface(SPI) Bus supported
  • Optional Socket for I2C NVSRAM (FM24V10) (129K x 8 bits)
  • Optional Socket for I2C Temperature Sensor (TMP102)
  • Optional Socket for I2C Encryption Processor (ATECC108)
  • Compatible with Mikrobus periperals
  • An Open-Source Hardware (OSHW) Reference Design

TSOC-ESP8266-EM Application Examples:

  • WiFi Sensor
  • WiFi Bridge
