thingSoC Software Architecture

  • Q: What is the thingSoC software architecture and API?
  • A thingSoC Compliant system is compatible with the Reference System Virtual Platform - Software Interface Standard (RSVP-SIS) an open source, vendor agnostic, application programming interface that has ports to several different IDE's.

    Please see the RSVP-SIS Organization Website for more information.
