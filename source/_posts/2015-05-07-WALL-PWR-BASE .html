thingSoC Reference Design
WALL-PWR-BASE is a an example PSoC 4200 Series, System on Chip, BASE module, a thingSoC reference design.

WALL-PWR-BASE includes a dual channel USB to Serial interface, along with an I2C EEPROM (internal) for storing the board configuration information, and status LED's. The WALL-PWR-BASE also includes VIN, USB, and Battery powered capabilities, with a single cell Li-Po battery charger. The battery can be charged from either VIN (i.e. 12 volt wall wart) or the USB input.

WALL-PWR-BASE Features :

  • Standard EEPROM for Car Model Identification
  • Standard DC Barrel Jack for 12V input
  • Battery (Single Cell Li-Po) Eliminator (supplies 4.2V)
  • USB Charging/Power Jack Output (i.e. 12V/Car Battery Adaptor)
  • Power Status LED's : Three (3)
  • Manual Reset Button
  • Real-Time Clock Oscillator (32,768 Hz)
  • Referance Clock Oscliaator (7.73728 MHz)
  • thingSoC BB-40-Double Form Factor

WALL-PWR-BASE is a simple, Wall Wart (12V brick) powered BASE implementation for powering thingSoC sockets. There are many ways to power thingSoC modules, this is the one you need if you want a higher voltage (12Vdc -15Vdc) input for powering your thingSoC modules. Complete design files are available using the links to the right.
