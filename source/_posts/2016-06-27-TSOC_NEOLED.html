The TSOC_NEOLED is a PSoC4 (Programmable System on Chip) based board with integrated NeoPixel (WS2812) driver, in an Embedded Module format for the new thingSoC, Internet of Things open standard.

TSOC_NEOLED - NeoPixel LED Driver Embedded Module :

The TSOC_NEOLED connects any thingSoC (or Mikrobus) module to strings of programmable color LEDs.

  • Supported by the Arduino IDE
  • thingSoC Compliant Module
  • Mikrobus Compatible Module
  • Support Inidividually addressable RGB, or RGBY Color LEDs

Example Applications :

An example of a portable color LED application, attached to our tandem bicycle for a Portland parade ride.
