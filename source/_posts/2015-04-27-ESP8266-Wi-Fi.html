thingSoC Reference Design
ESP8266-EXPA is a an example Wi-Fi Client/Access Point

ESP8266-EXPA includes an ESP8266-07 format Wi-Fi module, along with an I2C EEPROM for storing the board configuration information, and status LED's.

ESP8266-EXPA Features :

  • ESP8266-EXPA 802.11b/g Wi-Fi module
  • I2C Configuration EEPROM
  • Status LED's
  • thingSoC BB-20-Single Form Factor

Reference Design Status :

  • Untested PCB Layout, example schematic reference only.
