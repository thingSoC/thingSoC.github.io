thingSoC : Infineon XMC1200 CPU Base Board Reference Design

The TSOC-XMC1200 board is an Infineon XMC1200 CPU Base Board Reference Design. It provides a complete eight (8) pin full UART connection as well as the popular six (6) pin "FTDI" connector that is compatible with Arduino and many other development boards. The TSOC-XMC1200 board is based on the Infineon XMC1200, an ARM Cortex-M0 microcontroller.

thingSoC : ESP8266 Embedded Module

The TSOC-ESP8266-EM board is a complete Wi-Fi Device and Access point, supported by the Arduino IDE. It places the popular ESP8266 Wi-Fi System-on-Chip Device in a thingSoC Embedded Module Socket, and enables the ESP8266 to be used with any other thingSoC reference design, allowing it to be paired with a number of different host processors and peripherals.

thingSoC : USBUART (CDC) Reference Design

The TSOC-USBUART1 board is a single channel, USB to UART Base Board Reference Design. It provides a complete eight (8) pin full UART connection as well as the popular six (6) pin "FTDI" connector that is compatible with Arduino and many other development boards. The TSOC-USBUART1 board is based on the Cypress Semiconductor CY7C65213, a USB to UART Bridge Controller.

thingSoC : USB ENCORE3 Reference Design

The TSOC-ENCORE3 board is a USB to PSOC1 (Progammable System on Chip) Base Board Reference Design. It provides a complete thingSoC socket interface, including a full serial uart channel, I2C master channel, and SPI master channel with four (4) chip selects. The TSOC-ENCORE3 board is based on the Cypress Semiconductor CY7C64215, a USB Programmable System on Chip.

thingSoC : Open Source Sockets


thingSoC is an open source standard that defines a vendor independent socket system for the creation of new Internet Things; it addresses many of the limitations of current product offerings and standards, by adding capabilities such as automatic device discovery, device configuration, monitoring, instrumentation, and testing.

thingSoC works with many existing standards and Integrated Development Environments (IDEs), such as Arduino, rfDuino, Raspberry Pi, Beaglebone, X-Bee, and others; but adds features and capabilites that make building new IoT devices much easier.

The thingSoC virtual platform definition supports a number of different System-on-Chip platforms and peripherals, such as cell modems, packet radios, Wi-Fi, Bluetooth, and more.
