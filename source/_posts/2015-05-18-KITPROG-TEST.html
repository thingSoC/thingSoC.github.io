thingSoC Reference Design
KITPROG-TEST is a an example USB-to-UART,I2C and JTAG SWD Programmer, thingSoC reference design; utilizing a Cypress CY8C58xx, ARM Cortex-M3, System-on-Chip device.

KITPROG-TEST implements the standard PSoC Creator IDE USB based, device programmer and UART/I2C communications interface. It allows you to upload programs to blank PSoC ARM Cortex microcontrollers, as well as program blank I2C memories and other devices. The UART interface allows for a generic USB-to-Device serial channel for communicating with target devices.

KITPROG-TEST Features :

  • CY8C58xx : Cypress CY8C58xx, ARM Cortex-M3, System-on-Chip device
  • JTAG Target : Programming Header for SWD JTAG Interface (5 pin)
  • Reset Switch : Local Reset
  • Debug Switch : Target Debug/Reset
  • Debug LED : JTAG & Debug Mode
  • thingSoC : BB-40-Double Form Factor (Standard JTAG Pinout)
