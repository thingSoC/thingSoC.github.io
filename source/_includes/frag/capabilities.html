thingSoC includes several capabilities that the other small form factors do not support :

  1. Production Ready, embeddable hardware platforms that are easy to use
  2. High Speed Interface support, including SuperSpeed USB 3.0, PCI-E, and 10/100 Ethernet
  3. Standardized Mounting System, designed for high vibration platforms (robots, ebikes, drones)
  4. Built in support for battery, solar, and very low power applications
  5. Standardized auto-discovery firmware system, compatible with the Beaglebone Cape Manager
  6. Selectable connector sizing for very low pin count, cost sensitive applications
  7. Selectable physical sizing for cost trade-off based on PCB size (half-size, single, double, triple sizes)
  8. Standardized testing, instrumentation, monitoring, and data recording interfaces
  9. Interchangeable platform for break-out-boards, sensors, radios, and other devices
  10. Standardized cross-platform Application Programming Interfaces (APIs)
